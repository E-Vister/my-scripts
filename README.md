# My Scripts



Личный сборник мини-инструментов для повседневных задач и хобби.



## Инструменты



### Анализатор сценария (`/script-parser`)

Парсит текст сценария и возвращает JSON с репликами по персонажам.



**Формат входных данных:**

```
ДЕЙСТВУЮЩИЕ ЛИЦА:

ПЕРСОНАЖ А - @username

ПЕРСОНАЖ Б - @username


ПЕРСОНАЖ А: текст реплики

ПЕРСОНАЖ Б: текст реплики
```



**Результат:**

```json
{

  "ПЕРСОНАЖ А": ["текст реплики"],

  "ПЕРСОНАЖ Б": ["текст реплики"]

}
```
